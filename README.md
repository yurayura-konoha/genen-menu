# 🧂 減塩外食ナビ

外食チェーンを選ぶと、**1食の塩分目標内におさまるメニューの組み合わせ**を提案するWebアプリ(PWA)です。

**▶ アプリを使う: https://yurayura-konoha.github.io/genen-menu/**

スマホのブラウザで開いて「ホーム画面に追加」すると、アプリとして使えます(オフライン対応)。

## できること

- **組み合わせ提案** — お店と塩分目標(例: 1食2.5g)を選ぶと、目標内の注文パターンを提案。「みそ汁を残す」「つゆを残す」で収まる案や、目標+0.3g以内の「許容範囲」案も表示
- **栄養バランス** — 脂質エネルギー比を評価し「バランス○」「脂質多め」を表示
- **利用シーン対応** — カフェでは「食事」と「☕ お茶・間食」を分けて提案
- **1日トラッキング** — 食べたものを記録すると残り塩分を表示。コンビニ・自炊も自由入力で記録可能
- **週間ビュー** — 7日間の推移と週平均をグラフ表示
- **お気に入り** — よく頼む組み合わせをワンタップで呼び出し

## 対応店舗(19チェーン)

牛丼・丼(吉野家・すき家・松屋・なか卯)/定食(やよい軒・大戸屋)/麺類(丸亀製麺・リンガーハット)/カレー(CoCo壱番屋)/ファミレス(ガスト・サイゼリヤ・オリーブの丘)/バーガー(マクドナルド・モスバーガー・ケンタッキー)/カフェ(スターバックス・ドトール・コメダ珈琲店・星乃珈琲店)

## データについて

- 塩分・カロリー・脂質は原則として**各社公式の栄養成分公表値**に基づきます(取得時期はアプリ内の店舗選択時に表示)
- 一部店舗(ガスト・サイゼリヤ・丸亀製麺・カフェ各社)は公式の食塩相当量が公表されていないため**参考値(推定)**です。アプリ内で明示しています
- メニュー改定により数値は変わることがあります。正確な値は各社公式サイトでご確認ください

## 技術構成

- 単一HTML+Service Worker のPWA。サーバー・データベースなし
- 記録データはすべて端末内(localStorage)に保存され、外部送信されません
